# 🚀 FULL APP BUILD PROMPT (COPY & USE)

## 🧠 Prompt:

> Build a mobile-first full-stack marketplace + comparison app called **“PriceCompare”**.
> The app allows users to compare product prices from multiple vendors and choose the best option based on cheapest price, trust score, delivery speed, and proximity.
>
> The app must be inspired by a modern fintech + marketplace UI (clean, minimal, purple accent theme).

---

# 📱 1. TECH STACK

Use:

* Frontend: React Native (Expo) OR Next.js (mobile-first PWA)
* Backend: Node.js + Express
* Database: PostgreSQL or MongoDB
* Auth: Phone number OTP authentication only (no email/password)
* Styling: TailwindCSS or NativeWind
* State: Zustand or Redux Toolkit

---

# 🔐 2. AUTH SYSTEM (CRITICAL)

* Users sign up ONLY using phone number (+234 format support)
* OTP verification required
* No email login
* No password system
* Store session token (JWT)

User schema:

```
User {
  id
  name
  phone
  location
  role: "user" | "independent_seller" | "vendor"
  preferences: {
   best value: boolean
   budget friendly: boolean,
    trusted: boolean,
    nearby: boolean,
    fastDelivery: boolean
  }
}
```
